# 基于统计语言模型
## n-grams模型
所有长度为n的词序列与n-1的词序列在语料库中的相对概率连成的总和
可以理解为能计算出未知文本的泛化能力

n通常小于5

n-grams 语言模型是在n 阶马尔可夫假设下，对语料库中出现的长度为n 的词序列出现概率的**极大似然估计**

# 基于学习的语言模型
- 训练数据 training data  数量、质量 
- 假设类 Hypothesis Class （H）所有可能机器学习模型的集合 神经网络模型就是一个假设类 学习输出找到参数就是找到假设 
- 归纳偏置Inductive Bias 限制对模型假设进行选择。常用就是上下文关联 
- 学习范式 LLM通常是自监督学习范式 
- 学习目标 常用经验风险最小化ERM 
- 损失函数Loss Function 衡量模型在对应样本上的错误程度 基本上都是用连续函数来模拟0-1损失 
- 学习算法 旨在对损失进行优化 ———— 最优化理论 
- 泛化误差 目的在于减少泛化误差（generalization error） 
- PAC理论 概率近似正确 ~~ 当样本数量符合一定条件的时候，机器学习模型可以以一定的**概率**达到**近似正确** 

# RNN和transformer
上下文输入方式
1、串行输入RNN
2、并行输入transformer 一次性全部输入
## RNN 循环神经网络 Recurrent Neural Network
- 网络连接中包含环路 数学形式上就是时间维度上的**复合函数**
- Wh是RNN的返回循环的参数，特征值小于1和大于1时，由于连续乘积，容易导致梯度消失或者梯度爆炸
- LSTM解决梯度问题 把连乘变成状态累加 用遗忘门、输入门实现合理的累加 然后用输出门整合
- GRU将遗忘门和输入门进行合并
