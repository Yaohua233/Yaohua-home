# 基于统计语言模型
## n-grams模型
所有长度为n的词序列与n-1的词序列在语料库中的相对概率连成的总和
可以理解为能计算出未知文本的泛化能力

n通常小于5

n-grams 语言模型是在n 阶马尔可夫假设下，对语料库中出现的长度为n 的词序列出现概率的**极大似然估计**

## RNN 循环神经网络 Recurrent Neural Network
- 网络连接中包含环路 数学形式上就是时间维度上的**复合函数** 
- Wh是RNN的返回循环的参数，特征值小于1和大于1时，由于连续乘积，容易导致梯度消失或者梯度爆炸 
- LSTM解决梯度问题 把连乘变成状态累加 用遗忘门、输入门实现合理的累加 然后用输出门整合 
- GRU将遗忘门和输入门进行合并

# 语言模型的采样和评测  
## 采样 
需要根据概率向量从词表中选出本轮输出的词元
1、概率最大化方法 *最常见的文本，容易废话文学*
- 贪心搜索方法 都选择概率最大的词  *容易局部最优*
- 波束搜索法 每次保留b个可能性最高的词 找出其中b个词概率最高的组合 **维护所有可能的路径，计算其中联合概率最大的组合**
2、随机采样方法
先选出一组可能性搞的候选词，然后按照概率分布进行随机采样
- Top-k采样方法 *方差较大*和*方差较小*问题 **每次选前k个，重新归一化后随机采样，然后再进行下一步，每步都有不同可能**
- Top-p采样方法 设定阈值p来限制候选集 排除不合理的词元避免枯燥的情况
- temperature机制 **对随机性进行控制** 加一个参数来控制
## 评测 
1、内在评测 不依赖具体任务
- 困惑度（Perplexity PPL） 对概率的倒数开N次方*可以认为离奇的词出现的概率低，那么它的困惑度小* 困惑度减少意味着熵减，不会出现胡言乱语
2、外在评测 依赖具体任务
- 基于统计指标 BLEU（BiLingual Evaluation Understudy） 多层次的n-gram匹配精确度的几何平均 **关键在于统计匹配** *对语义的泛化不好*
- 基于语言模型
  - 基于上下文嵌入 BERTScore 有三个指标：精度Precision，召回Recall，F1
  - 基于生成模型 用另一个大模型去评价大模型生成内容的质量
